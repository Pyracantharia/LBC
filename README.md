# LBC
LeBonCoin de la M2L

Cahiers des charges:
-Pouvoir créer un compte de manière sécurisée
-Pouvoir se connecter de manière sécurisée
-Pouvoir créer une annonce
-Pouvoir modifier une annonce
-Pouvoir supprimer une annonce
-Voir la liste des annonces
-Voir le détail d'une annonce
-Pouvoir filtrer les annonces selon plusieurs critères
-Pouvoir mettre des annonces dans ses favoris
-Pouvoir envoyer un message concernant une annonce
-Pouvoir consulter ses messages
-Voir les annonces les plus consultées
-Barre de recherche en AJAX
